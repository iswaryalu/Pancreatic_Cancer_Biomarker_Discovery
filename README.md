# Integrative Co-expression and Machine Learning Analysis for Early Non-Invasive Detection of Pancreatic Cancer

## Overview

This project combines differential gene expression analysis, WGCNA, pathway enrichment, and machine learning techniques to identify robust biomarkers for early non-invasive detection of pancreatic cancer.

## Objectives

- Identify differentially expressed genes
- Construct co-expression networks using WGCNA
- Perform functional enrichment analysis
- Select diagnostic biomarkers using:
  - LASSO
  - SVM-RFE
  - Random Forest
- Validate biomarkers in tumor and blood datasets

## Key Findings

Five-gene biomarker panel:

- AHNAK2
- TRIM29
- SLC6A14
- ITGB4
- CTSE

## Tools

- R
- WGCNA
- limma
- glmnet
- randomForest
- e1071
- GEO datasets
